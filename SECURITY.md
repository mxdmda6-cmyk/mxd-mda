# Security Policy

**Report a vulnerability:** Email security@mxd-mda.example (or open a private security advisory).  
Do **not** create public issues for sensitive reports.

## Supported Versions
We generally support the latest `main` and the most recent tagged release.

## Disclosure
- We aim to acknowledge within 2 business days.
- We will confirm, triage severity, and propose a fix or mitigation.
- Credit is optional and will be given upon request after a patch is available.
