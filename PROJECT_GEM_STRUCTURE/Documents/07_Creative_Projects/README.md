- `MXD_MDA_Logo_Primary_v3_2025-11.png`

**Version Control:**
- Start at `v1`
- Increment for each major revision
- Use `_Draft` for work-in-progress
- Use `_Final` only when truly finished
- Archive old versions to `99_Archive/Old_Projects/`

## ğŸ’¡ Creative Organization Tips

### 1. **Separate Work-in-Progress from Finals**
Create temporary `_WIP/` folders in each project for active work.
Move to final location when done.

### 2. **Use Version Numbers Religiously**
Don't end up with:
```
logo_final.png
logo_final_v2.png
logo_final_FINAL.png
logo_final_USE_THIS_ONE.png
```

Instead:
```
MXD_MDA_Logo_v1.png
MXD_MDA_Logo_v2.png
MXD_MDA_Logo_v3_Final.png
```

### 3. **Archive Aggressively**
If you haven't touched it in 3 months, archive it.
Keep active projects lean and focused.

### 4. **Link Related Files**
Create a `_LINKS.txt` file in project folders:
```
Related Files:
- Brand Guide: [Google Drive link]
- Marketing Plan: [Google Drive link]
- Content Calendar: [Google Sheets link]
- Notion Dashboard: [Notion link]
```

### 5. **Regular Project Reviews**
Set calendar reminder:
- Weekly: Review each project's status
- Monthly: Archive completed work
- Quarterly: Assess priorities, pivot if needed
