### Step 2: Generate QR Code

**Option A: Online Generator (Easiest)**
1. Go to [qr-code-generator.com](https://www.qr-code-generator.com/)
2. Select "URL" type
3. Paste your GitHub Pages URL
4. Customize:
   - Frame: "Scan Me" or "Happy Birthday"
   - Colors: Match Day of the Dead theme (orange/purple/pink)
   - Logo: Upload a small sugar skull image (optional)
5. Download high-resolution PNG (at least 1000x1000px)

**Option B: QR Code Monkey (More Customization)**
1. Go to [qrcode-monkey.com](https://www.qrcode-monkey.com/)
2. Enter your URL
3. Customize colors:
   - Body color: #9D4EDD (purple)
   - Eye color: #FFA500 (orange)
4. Add logo/image (optional)
5. Select high quality (300 DPI for printing)
6. Download PNG

**Option C: Canva Integration**
1. Generate QR code using one of the methods above
2. Open Canva
3. Create new design: "Card (5 x 7 in)"
4. Upload your QR code
5. Add design elements:

## ğŸ”§ Troubleshooting

**QR Code Won't Scan?**
- Ensure high contrast (dark on light background)
- Increase size
- Reduce customization/logos
- Test different QR generators
- Check URL is correct and accessible

**Card Not Loading?**
- Verify GitHub Pages is enabled
- Check URL is publicly accessible
- Test on different devices/browsers
- Ensure no typos in URL
